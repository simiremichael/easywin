## Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd easywin
```

2. Install dependencies:
```bash
npm install
```

## Running Locally

1. Start the development server:
```bash
npm run dev
```

2. Open your browser and navigate to:
```
http://localhost:5173
```

## How to Play

1. Click "Play Now" on the home screen
2. Scratch any 3 cards by clicking or dragging on them
3. Match 3 identical symbols to win prizes
4. Click "Play Again" to start a new game

## Build for Production

```bash
npm run build
```
